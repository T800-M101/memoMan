## 📋 Description

**MemoMan** It's a simple, lightweight, and free API client designed to make HTTP requests easily and intuitively. This project originated as a personal tool for developers who need to test REST endpoints without complications or costs.

Perfect for:
- ✅ Testing APIs during development
- ✅ Debugging REST endpoints
- ✅ Share collections of requests
- ✅ Learn how HTTP requests work

## ✨ Features

- 🎯 **Clean & Modern Interface** - Designed to be intuitive
- 📦 **Organized Collections** - Group your requests by projects
- 🔑 **Authentication Support** - Bearer Token included
- 📊 **Parameters & Headers** - Complete request configuration
- 📝 **JSON Editor** - For your request body
- 📥 **Response Visualization** - With syntax highlighting
- 🎨 **Light/Dark Theme** - Comfortable for long sessions
- 📱 **Responsive** - Works on desktop and mobile devices

## 🚀 Installation & Usage

### Prerequisites

- Node.js (version 18 or higher)
- npm (version 9 or higher)

### Installation Steps

```bash
# 1. Clone the repository
git clone https://github.com/T800-M101/memoMan.git

# 2. Enter the directory
cd memoman

# 3. Install dependencies
npm install

# 4. Start the development server
npm start
# or
ng serve
